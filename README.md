## 📌 Project Overview
Developed as a comprehensive hardware design project, this device generates sine, square, triangular, and sawtooth waveforms with a highly tunable frequency range spanning from 20 Hz to 20 kHz. To maximize versatility, the square wave output features a finely adjustable duty cycle ranging from 1% to 99%.

## ⚙️ Core Architecture & Circuitry
The system architecture relies on precision analog components and op-amp feedback configurations to synthesize clean waveforms.

* **Triangular & Sawtooth Waves:** Engineered utilizing a Schmitt trigger cascaded with an integrator.
* **Square Wave:** Generated by feeding the triangle wave into a comparator, allowing for precise duty cycle variation.
* **Sine Wave:** Produced using a highly stable Wien Bridge Oscillator circuit.
* **Signal Integrity:** Utilizes a common inverting amplifier stage to maintain consistent amplitude levels across all selected waveforms.
* **Push-Pull Output Buffer:** Features a complementary NPN-PNP transistor pair paired with an op-amp buffer to completely eliminate the crossover distortion typically found in standard push-pull amplifiers.

---

## 🛠️ The Tech Stack
* **Circuit Simulation:** National Instruments (NI) Multisim.
* **PCB Design & Fabrication:** Altium Designer (Manufactured via JLCPCB). Includes a dedicated Main PCB and a Power PCB for 230V to 12V conversion.
* **3D Modeling & Enclosure:** Fusion 360, focusing on a highly ergonomic and user-friendly interface.

## 🚀 Future Roadmap (V2: The Digital Upgrade)
While V1 is purely analog, the next iteration of this project will bridge the hardware with an embedded IoT stack. 

* **Microcontroller Integration:** Integrating an **ESP32** module to add digital smart features.
* **Real-Time Display:** Adding an OLED/LCD screen to output real-time waveform characteristics, including live frequency and amplitude data.
* **Custom Power Distribution:** Designing a secondary step-down PCB circuit to safely convert the 12V system power to 3.3V to drive the ESP32.
* **PCB Optimization:** Migrating all variable resistors and external switches directly onto a unified PCB to eliminate internal wiring and optimize signal noise.
